Class: Buffer

Tone. Buffer

Buffer loading and storage. Tone.Buffer will load and store the buffers in the same data structure they were given in the argument. If given a string, this.buffer will equal an AudioBuffer. If constructed with an array, the samples will be placed in an array in the same order.
