Class: Filter

Tone. Filter

Filter object which allows for all of the same native methods as the BiquadFilter (with AudioParams implemented as Tone.Signals) but adds the ability to set the filter rolloff at -12 (default), -24 and -48.

new Filter(freq, type, rolloff)

Parameters:
Name Type Argument Default Description
freq number | Object <optional>
 350

the frequency
type string <optional>
 lowpass

the type of filter
rolloff number <optional>
 -12

the rolloff which is the drop per octave. 3 choices: -12, -24, and -48
