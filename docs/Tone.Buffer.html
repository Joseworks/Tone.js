Class: Buffer

Tone. Buffer

Buffer loading and storage. Tone.Buffer is used internally by all classes that make requests for audio files such as Tone.Player, Tone.Sampler and Tone.Convolver .

Aside from load callbacks from individual buffers, Tone.Buffer provides static methods which keep track of the loading progress of all of the buffers. These methods are onload, onprogress, and onerror.

<static> onerror()

Callback if one of the buffers in the queue encounters an error. The error is passed in as the argument.

Example
Tone.Buffer.onerror = function(e){
	console.log("there was an error while loading the buffers: "+e);
}

<static> onload()

callback when all of the buffers in the queue have loaded

Example
//invoked when all of the queued samples are done loading
Tone.Buffer.onload = function(){
	console.log("everything is loaded");
};

<static> onprogress()

Callback function is invoked with the progress of all of the loads in the queue. The value passed to the callback is between 0-1.

Example
Tone.Buffer.onprogress = function(percent){
	console.log("progress:" + (percent * 100).toFixed(1) + "%");
};
