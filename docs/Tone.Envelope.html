Class: Envelope

Tone. Envelope

ADSR envelope generator attaches to an AudioParam or Signal.

new Envelope(attack, decay, sustain, release)

Parameters:
Name Type Argument Default Description
attack Tone.Time | Object <optional>
 0.01

the attack time in seconds
decay Tone.Time <optional>
 0.1

the decay time in seconds
sustain number <optional>
 0.5

a percentage (0-1) of the full amplitude
release Tone.Time <optional>
 1

the release time in seconds
Example
 var gainNode = Tone.context.createGain();
 var env = new Tone.Envelope({
 	"attack" : 0.1,
 	"decay" : 0.2,
 	"sustain" : 1,
 	"release" : 0.8,
 });
 env.connect(gainNode.gain);
