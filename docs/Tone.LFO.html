Class: LFO

Tone. LFO

The Low Frequency Oscillator produces an output signal which can be attached to an AudioParam or Tone.Signal for constant control over that parameter. the LFO can also be synced to the transport to start/stop/pause and change when the tempo changes.

new LFO(frequency, outputMin, outputMax)

Parameters:
Name Type Argument Default Description
frequency Tone.Time <optional>
 "4n"
outputMin number <optional>
 0
outputMax number <optional>
 1
Example
 var lfo = new Tone.LFO("4n", 400, 4000);
 lfo.connect(filter.frequency);

The amplitude of the LFO, which controls the output range between the min and max output. For example if the min is -10 and the max is 10, setting the amplitude to 0.5 would make the LFO modulate between -5 and 5.

syncFrequency()

Sync the signal to the Transport's bpm. Any changes to the transports bpm, will also affect the oscillators frequency.

Inherited From: Tone.Oscillator#syncFrequency
