Class: FMSynth

Tone. FMSynth

the FMSynth is composed of two MonoSynths where one MonoSynth is the carrier and the second is the modulator.

new FMSynth(options)

Parameters:
Name Type Description
options Object

the options available for the synth see defaults below
Example
 var fmSynth = new Tone.FMSynth();

The modulation index which is in essence the depth or amount of the modulation. In other terms it is the ratio of the frequency of the modulating signal (mf) to the amplitude of the modulating signal (ma) -- as in ma/mf.
